# Veritas Technical Diagrams

Technical diagram assets for the Veritas narrative.

- `excalidraw/` contains editable Excalidraw source files.
- `svg/` contains rendered SVG exports with matching numbered filenames.

## Files

- `01-tech-stack.excalidraw` - Layered product and infrastructure stack, from edge SDK to governance console.
- `02-production-architecture.excalidraw` - End-to-end runtime architecture across customer devices, bank tenancy, control plane, registry, and console.
- `03-federated-round-flow.excalidraw` - A signed federated learning round with robust aggregation, differential privacy, and transparency logging.
- `04-data-sovereignty-boundaries.excalidraw` - Explicit trust boundaries showing which artifacts cross and which data never leaves its owner.
- `05-demo-runtime-flow.excalidraw` - Demo execution path for scale simulation, siloed baseline, Veritas comparison, metrics, and audit proof.
- `06-threat-and-governance-flow.excalidraw` - Abuse cases, defenses, governance outcomes, and audit artifacts.
- `07-bank-onboarding-deployment.excalidraw` - Adoption path from sealed-node deployment through shadow mode and go-live.

Open any file in `excalidraw/` to edit the source. Use the matching SVG in
`svg/` when embedding a rendered diagram in docs or slides.
